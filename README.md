# Watch Heart

Windows 桌面心率接收器，用于连接 iPhone 上提供标准 BLE Heart Rate Service 的“心率广播”应用。包含游戏悬浮条、实时天气、自动定位、心率趋势和同步歌词。

## 使用

1. 在 Apple Watch 打开“心率广播”并开始测量。
2. 在 iPhone 打开“心率广播”，允许蓝牙和健康数据权限。
3. 启动 Watch Heart，点击“扫描并连接”。
4. 从扫描结果中选择 iPhone 广播设备。

## 游戏悬浮条

- 主界面可分别调整悬浮条缩放比例和宽度。
- `Ctrl+Shift+H`：显示或隐藏悬浮条。
- `Ctrl+Shift+L`：强制取消鼠标穿透。
- `Ctrl+Alt+Shift+L`：备用取消穿透快捷键。
- 天气每 15 分钟自动更新，也可使用 Windows 定位获取当前城市。
- “低占用游戏模式”会关闭背景模糊、阴影和过渡效果，并降低歌词读取频率。

## 歌词来源

歌词按以下优先级显示：

1. **汽水音乐播放器直读**：直接读取汽水音乐“桌面歌词”页面当前行，不截图、不进行 OCR。
2. **在线同步歌词备用**：播放器直读未连接、没有当前行或被关闭时，根据 Windows 媒体会话信息从 LRCLIB 匹配。
3. **手动匹配**：可在主界面输入歌曲名和歌手。

### 汽水音乐播放器直读

1. 开启“优先使用汽水音乐播放器直读”。
2. 首次使用时完全退出汽水音乐，然后点击“安装/重连”。
3. 在汽水音乐中开启“桌面歌词”。
4. 汽水音乐升级后如果直读失效，重复第 2 步。
5. 可开启“自动隐藏汽水桌面歌词”；原歌词页面会保持运行和同步，但不再显示或拦截鼠标。关闭开关或退出 Watch Heart 后会自动恢复。

汽水音乐 3.5.1 的自定义外壳会阻止外部 CDP 目标，因此程序使用作用相同但更稳定的播放器页面本地桥接。首次启用时会为 `desktopLyrics.asar` 创建原文件备份并安装一个很小的歌词发送脚本。本地桥只监听 `127.0.0.1:19228`，只发送当前一行文字；不会获取窗口图像，也不会持续枚举或截取桌面窗口。汽水音乐断开时，悬浮条会自动回退到在线时间轴歌词。
